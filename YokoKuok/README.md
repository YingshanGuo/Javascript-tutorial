# Profile
> Target：使用Webpack提升前端开发效率  
> Techs：NodeJs + Npm + React + Babel + Webpack + Less

### Features
* 支持 ES6
* 支持 React
* 支持 jQuery
* 支持 Less
* 支持 Font-Awesome

### Usage
> 1、安装依赖：$ npm install  
> 2、启动应用：$ npm start  
> 3、静态编译：$ npm run build
